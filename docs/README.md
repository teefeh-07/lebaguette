# Le Baguette NFT Collection

A French-themed NFT collection built on the Stacks blockchain using Clarity smart contracts.

## Overview

Le Baguette is an artisan NFT bakery that creates unique digital baguettes with various French-inspired attributes on the blockchain.

## Features

- **NFT Minting**: Mint unique baguette NFTs with randomized attributes
- **Whitelist Support**: Special pricing for whitelisted addresses
- **Dynamic Attributes**: Each baguette has unique crust, filling, length, flavor, and rarity
- **Trait System**: Boolean traits including organic, gluten-free, and seeds
- **Royalty System**: 5% royalties on secondary sales
- **Wallet Integration**: Support for Stacks wallets via @stacks/connect
- **WalletConnect**: Cross-platform wallet connectivity
- **Chainhooks**: Real-time event monitoring for mints and transfers
